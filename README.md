# DesignVerse

DesignVerse is a premium design platform for exploring color palettes, design principles, and inspiration in a premium space-themed environment.

## Project info

**URL**: https://designverse.app

## How can I edit this code?

If you want to work locally using your own IDE, you can clone this repo and push changes.

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd DesignVerse

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

You can deploy this project using any modern static hosting platform (Vercel, Netlify, Cloudflare Pages, etc.).

1. Run `npm run build` to create a production bundle.
2. Upload the contents of the `dist` folder to your hosting provider.
